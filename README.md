# BoxrecWebScraper

Currently this project only supports Boxrec

## NOTE: Project will not work without a credentials file for the scraper

-  Please ensure that within the `/scraper` directory of the project, you add a json file called `credentials.json`
-  Within the file add the following structure:

```
{
   "username": "<ENTER-YOUR-USERNAME>",
   "password": "<ENTER-YOUR-PASSWORD>",
}
```
